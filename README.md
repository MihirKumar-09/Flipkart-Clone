# 🛒 Flipkart Clone – MERN Stack E-Commerce Application
A full-stack e-commerce web application inspired by Flipkart, built using the MERN stack.
This project replicates core e-commerce functionalities including product listing, authentication, cart management, and order-related features.

⚠️ Currently optimized for desktop view. Responsive design improvements are in progress.

# 🚀 Live Demo
Link : https://flipkart-clone-1-eay9.onrender.com/
The project is hosted on the free tier of Render, where the backend service may go into sleep mode after inactivity.
